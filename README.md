# Predicting-Chronic-Disease-using-ML
## Overview
This project predicts Diabetes, Heart Disease, and Kidney Disease using ML models.

## Technologies Used
- Python
- Scikit-learn
- TensorFlow
- Flask

## Models
- CNN (Diabetes)
- Gradient Boosting (Heart Disease)
- XGBoost (Kidney Disease)

## Results
- Accuracy: 92%
